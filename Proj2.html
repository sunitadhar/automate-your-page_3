Lesson 1: Basics of the Web and HTML

The Web

The web is a group of computers that communicate with each other. When a person browses on a computer; the request is send to the server through the Internet. The server finds the page and sends back to the persons computer screen.

HTML (Hypertext Markup Language)

It is the language used to create most of the web pages. It describes the text contents and how the page should look, and how to inclued images, links and all.

Elements and Tags

HTML documents are made of HTML elements. When writing HTML, we use the tags to tell the browser the type of element we are using. Example: element is to make the text bold. element is to italicizes the text.

Inline and Block Elements

HTML elements are written in either inline or blocks. Blocks are like boxes that are invisible in the browser.

Box Model

The box model is used to position boxes on a web page and consists of:

Margin: Space between boxes on a page and is invisible.

Border: Outline of a box.

Padding: Area around the center of the box.

Content: The actual contents of the box (like nucleus).

Boxes can be sized in three ways: size by pixels, size by using a percent value, or by using "box-sizing: border-box" for every element

Text Editors

Lesson 3: CSS Style

Avoiding Repetition and Errors

While programming, we may have to use certain style attributes to different HTML elements for the web page. We can do this by writing the same style attribute for different HTML elements again and again. It means repetition and will lead to errors. We can avoid the errors and repetition by assigning the same class to different <div> and then specify the style for that class at one place. Any future changes will be easy too! For example if we want to change text color on the web page; meaning we have to change the text color attribute at all the places in all the <div>, while as if we have specified a same class for the different <div>, we just need to change the text attribute at one place only.

CSS(Cascading Style Sheets)

Programmers use it to give style to HTML elements. We can use CSS to use different Background colors, use different colors in different blocks, make text bold, change font, italicized and so on.. There are three ways we can do this:

Method 1. At the top of the HTML, we can add head element and write the CSS under the style element.

Method 2. It lets us to stay organized. We can create a separate file with CSS code and link that to the HTML file. We can include the style file name in the head element of the HTML file.

Method3. We write the style inline with HTML. This method should be avoided as it results in lot of repetition and errors.

Lesson 4:Telling Computers what to Do

Computer

Most machines are designed to do few things, but computers cannot do anything on its own, unless you write programs to instruct them to do many things. Computer is an electronic device for storing and processing data according to instructions given to it in a program.

Computer Program

Program is a set of instructions that programmers write to instruct computer to what to do. Some examples are Web browsers, games, apps and so on.

Programming Language

Programming Language, like Python is a language that programmers use to write set of instructions that in turn instructs the computer to what to do.

Python

Python is an programming language. It converts the code into the language computer understands and executes. .

Python Grammar

Python Grammar are set of rules that programmers follow while writing the code. It is the same as we have English grammar rules. In Python, we have expression, followed by operation, followed by expression. Expression can be replaced either by another expression or numbers (0,1,...). Operation is what kind of operations we want to do with those numbers. In short, we have to follow set of rules while coding so that computer can interpret and execute correctly.

Variables in Python

We use variables to give names to values. For example if speed_of_car is a variable with a value of 20, then the following code would print out 2000:

   print 100 * speed_of_car.

We can assign the value 20 to the variable speed_of_car by writing code like this:

   speed_of_car = 20

We can even change the value of a variable by re-assigning it to a different value later. Use of variables in programming improves code readability, and can store the value of important data that can be easily changed.

How we use '=' in Python is also different: If we type 1 + 2 = 3, the equals sign means "is the same as"; while as if we type speed_of_car = 20, the equals sign means "takes the value of".

Strings in Python

String is a sequence of characters in a quote or quotes.
For example:

   name = 'Dave'
   print 'Hello ' + name

The result will be Hello Dave

One more important thing in Python is: 2 is a number while "2" is a string too. The code 2 + 2 would give 4. The code "2" + "2" would give "22".

Functions or Procedures

A function takes an input, processes it and then produces an output. We can code this by using the keyword def and then writing fuction name followed by the function parameters in paranthese. In the next line, we write the code that specifies what to do with the input parameters. When this procedure is called, the parameters are replaced by actual values. For example: A procedure named add might take two numbers as input and returns the sum of two numbers as output and can be written as follows:

def add( x, y )
  return x + y

When we call this procedure as:

print add( 2, 3 ) ;
It will give us the output of 5.

Without the return statement, we cannot see the output of 5. It will give us the output of NONE.
Once a procedure is defined, we never have to define it again and we can reuse them forever. It helps programmers to avoid repetition too!!

Loops

While loop: A while loop statement in Python programming language repeatedly executes a single statement or a block of statements till test expression is true.
Syntax: while <test expression>
                <block>
When the test expression becomes false, program control passes to the line immediately following the loop. The while loop might not ever run. When the condition is tested and the result is false, the loop body will be skipped and the first statement after the while loop will be executed.
Example: The following while loop will print the numbers from 0 to 10.

i = 0
while i < 10:
   print i
   i = i + 1

Infinite Loop: A loop becomes infinite loop if a condition never becomes false. Example: Since i is never going to be equal to 10; the following while loop will run for ever or till computer memory gives up.

i = 1
while i != 10:
   i = i + 2
   print i

Structured Data

We learned two ways of structuring data: Strings and Lists.

Strings: are the most popular types in Python. We can create them by enclosing sequence of characters in quotes and assigning it to the variable. Python treats single quotes the same as double quotes. For example:

  s = 'yabba!'

Lists: It is a most versatile datatype available in Python. It can be written as a list of values separated by commos between square brackets. Important thing about a list is that value can be sequence of anything like strings, numbers and so on. Lists can even contain other lists. For example:

  p = ['y', 'a', 'b', 'b', 'a', '!', '9', [1, 3]]

Accessing Values in strings and Lists is similar. Each element or value of a sequence is assigned a number - its position or index. The first index is zero, the second index is one, and so forth. To access values in strings or lists, we use the square brackets indicating the index to obtain value available at that index. For example:

  s[0] = p[0] = 'y'
  s[2:4] = 'bb'
  p[2:4] = [ 'b', 'b']

Mutation: It means adding or changing the same variable space without creating the new variable space. Like in lists we can append to the same variable and not create a new variable like string and adding or changing the new data.

List operations:
Append: We can add to elements in a list with the append operator, like <list>.append(<element>). The + operator can also add a element in a list, but it creates a new variable instead of appending to the same variable.

Len: We can use len operator to find no. of elements in a list, like len(<list>)

Index and In: We can use these operations to find the index number of a particular cell in a list.

For Loops: We can use for loops to go thru the values in the list using one statement and not needing to use the index of that cell. Like: for<name> in <list> For example:
If we have to print all the numbers stored in list p, we can use for loop as follows:

  for e in p:
     print e

The above code will go thru all the values stored in list p and print them!
